# AYANAMI

一座只属于绫波的数字展馆。新版以 React、Vite、Motion 和响应式 AVIF/WebP 图像构建。

## 本地运行

```bash
npm install
npm run dev
```

生产构建与预览：

```bash
npm run build
npm run preview
```

`npm run build` 会先从 `artwork-source/` 重新生成 `public/assets/` 下的响应式图片。旧站原文件完整保存在 `artwork-source/legacy-site/`，生成的首页原图保存在 `artwork-source/generated/`。
